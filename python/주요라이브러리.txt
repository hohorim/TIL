- WSGI(Web Server Gateway Interface): 파이썬 웹 어플리케이션과 웹 서버 간의 인터페이스로서 사용.
	이는 동기적인 프로그래밍 모델을 따르기 때문에 동시 처리에 제약이 있음.
	Flask, Django 와 같은 프레임워크가 WSGI를 기반으로 동기처리를 지원
	
- ASGI(Asyncronous Server Gateway Interface): 비동기 웹 어플리케이션을 위한 파이썬 웹 서버와 웹 프레임워크 간의 인터페이스. 비동기 처리 지원.
	FastAPI 같은 프레임워크

- uvicorn : 비동기 인터페이스를 통해 구현된 ASGI 웹 서버

- apscheduler: 스케쥴 관리 라이브러리

- dramatiq: 파이썬 어플리케이션에서 사용가능한 비동기 처리 솔루션
	동시성을 제어할 수 있는 기능 옵션을 제공
	
	2개의 요청이 동시에 들어오면 먼저들어온 요청은 dramatiq 의 Retries 미들웨어에 의해 오류가 발생하여 설정한 큐에 들어가고, 다음 시도를 기다린다.
	이런식으로 동시성을 해결함.
	그래서 보통 메시지 큐 역할을 하는 미들웨어를 dramatiq 앞에 놈
