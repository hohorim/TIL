- flanner
	┌─────────────────────────────────────┐
	│  Physical Network (L2/L3)           │
	│  ┌───────────────────────────────┐  │
	│  │  VXLAN Tunnel (Overlay)       │  │
	│  │  ┌─────────────────────────┐  │  │
	│  │  │  Pod Network            │  │  │
	│  │  │  (캡슐화됨)              │  │  │
	│  │  └─────────────────────────┘  │  │
	│  └───────────────────────────────┘  │
	└─────────────────────────────────────┘

	패킷 흐름:
	Client (1.2.3.4) → Node A
	  → [캡슐화] Outer: Node A IP → Node B IP
		 Inner: Client IP → Pod IP
	  → Node B [탈캡슐화]
	  → [SNAT] Node IP → Pod
	  → Pod가 보는 IP: Node IP (원본 손실)
  
  
- calico
	┌─────────────────────────────────────┐
	│  Physical Network (L3 Routing)      │
	│  ┌───────────────────────────────┐  │
	│  │  Direct IP Routing            │  │
	│  │  (캡슐화 없음)                 │  │
	│  │  ┌─────────────────────────┐  │  │
	│  │  │  Pod Network            │  │  │
	│  │  │  (직접 라우팅)           │  │  │
	│  │  └─────────────────────────┘  │  │
	│  └───────────────────────────────┘  │
	└─────────────────────────────────────┘

	패킷 흐름:
	Client (1.2.3.4) → Node A
	  → [직접 라우팅] Client IP → Pod IP
	  → Node B (라우팅 테이블 참조)
	  → Pod가 보는 IP: Client IP (원본 유지!)
